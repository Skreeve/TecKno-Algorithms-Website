Binary Search Algorithm

Binary Search is a fast searching algorithm that works on the principal of divide and conquer. For a binary search to work, it is very important for the array to be sorted in an ascending order.

Example:

Below is a sorted array and the algorithm will be searching for the number 57.
(9 11 12 20 20 27 28 29 37 40 40 57 74 89 92 99)

First Pass:

The algorithm identifies the first index on the array which is assigned to the variable called low,
while the last index on the array is assigned to the variable high .
The variable mid will mark as the middle of the array. To get the mid, we will use the formula mid = (high+low)/2
In this case, low = 0 and high = 15. Using our formula mid = (high + low)/2, (15+0)/2 = 7.5 (The algorithm is going to use the integer value(7), so we are not rounding up to the nearest whole integer.)

Second Pass:

The algorithm checks if the number at the mid index position (29) is equal to the number being searched (57).

Third Pass:

Since the number at the mid index is less than the number being searched for, we then move the variable low to the index position mid+1 ( so index position 8) and we discarded the left hand side of the array.
So we are discarding this part of the array (9 11 12 20 20 27 28 29)
and focusing on this part of the array (37 40 40 57 74 89 92 99).

Forth Pass:

So the algorithm repeats the process of finding the mid value by using the formula mid =(high+low)/2.
In this case the low variable is assigned to index position mid+1(so index position 8) and the high variable is the last index position. Mid = ( 15 + 8)/ 2 = 11.5 (The algorithm is going to use the integer value(11)).

The algorithm checks if the number at the mid index position (57) is equal to the number being searched (57). Now the algorithm has found the number 57.
