Linear Search Algorithm

A linear search is the most basic method of searching an array/dataset.

In an unsorted array, the element you are looking for could be located anywhere within the array, so there is no efficient way to search for it. It is therefore best to take a sequential approach. Linear search will start at the beginning and search through each item in the list until it reaches the end.

Here is simple approach is to do Linear Search (As shown in the trinket bellow):

  • Start from the leftmost element of array and one by one compare the element we are searching for with each element of the array.
  • If there is a match between the element we are searching for and an element of the array, return the index postion that the item was found.
  • If there is no match between the element we are searching for and an element of the array, return -1.
