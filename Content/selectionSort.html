Selection Sort Algorithim

Selection sort places the smallest item in the first position in an ascending list, and the largest item in the last place.

Example: (64 25 12 22 11)

First Pass:

Find the minimum element between [0-4] and swap it with the element at the begining of the list.

(64 25 12 22 11) -> (11 25 12 22 64)

Second Pass:

Find the minimum element between [1-4] and swap it with the element at [1].

(11 25 12 22 64) –> (11 12 25 22 64)

Third Pass:

Find the minimum element between [2-4] and swap it with the element at [2].

(11 12 25 22 64) –> (11 12 22 25 64)

Forth Pass:

While the array is already sorted, our algorithm does not know if it is completed. So it still compares between [3-4] to find which is the minimum element between the two.

(11 12 22 25 64) –> (11 12 22 25 64)
